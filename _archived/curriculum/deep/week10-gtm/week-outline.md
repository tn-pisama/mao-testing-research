## Design Partner Criteria (Updated 2025)

**Ideal Design Partner:**
- [ ] Running multi-agent in production (or within 3 months)
- [ ] Using LangGraph, CrewAI, AutoGen, or any of the 7 MAST frameworks
- [ ] 3+ agents coordinating
- [ ] Experienced agent failures
- [ ] Technical decision maker accessible
- [ ] Willing to share feedback regularly
- [ ] Can provide testimonial if successful
- [ ] **NEW: Interested in adversarial robustness**
- [ ] **NEW: Values academic-backed methodology**

**Red Flags:**
- "We're just exploring AI"
- No production timeline
- Single-agent only
- Build-everything-internally culture
- No clear pain point
- **NEW: "We're waiting for FAANG to solve this"**
- **NEW: Extreme patent/IP concerns (may need legal clarity first)**
