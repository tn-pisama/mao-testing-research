# MAO Testing Expert Curriculum: Deep Edition

**Duration:** 10 weeks (50 days)
**Depth:** 5x the accelerated curriculum
**Goal:** World-class expertise for technical founder credibility

| Week | Topic | Days | Deliverables |

| 1-2 | LangGraph Mastery | 10 | 3 production-grade implementations |
| 3-4 | CrewAI Mastery | 10 | 3 production-grade implementations |

## What "5x Depth" Means

| Accelerated (10 days) | Deep (50 days) |
|----------------------|----------------|
| Concept overview | Implementation details |
| 1 code example | 10+ code examples |
| "Here's what it does" | "Here's how to build it" |
| Quick reference | Complete reference |
| Know the terms | Teach the terms |
| Awareness | Mastery |

## Weekly Breakdown

### Weeks 1-2: LangGraph Mastery
- Complete StateGraph API
- All checkpoint implementations
- Human-in-the-loop patterns (5 variations)
- Streaming architectures
- Subgraph composition
- Error handling patterns
- Performance optimization
- Production deployment
- **Projects:** Research agent, Code reviewer, Customer support bot

### Weeks 3-4: CrewAI Mastery
- Agent design patterns (10 archetypes)
- Task dependency graphs
- Memory systems (short/long/entity)
- Delegation strategies and anti-patterns
- Flows vs Crews decision framework
- Custom tool creation
- Enterprise integration
- **Projects:** Content pipeline, Research team, Code review crew

### Week 5: AutoGen & Emerging Frameworks
- GroupChat internals
- Code execution sandboxing
- Agent selection algorithms
- Microsoft Agent Framework migration
- OpenAI Swarm analysis
- Framework selection decision tree
- **Project:** Same system in 4 frameworks

### Weeks 6-7: MAST Taxonomy Deep Dive
- All 14 failure modes with detection algorithms
- Academic papers review (5 key papers)
- Building automated detectors
- Classification confidence scoring
- Root cause analysis automation
- **Project:** MAST detector library

### Week 8: Testing Methodology
- Property-based testing with Hypothesis
- Statistical testing theory
- Golden dataset curation methodology
- LLM-as-Judge calibration
- Regression testing architecture
- CI/CD pipeline design
- **Project:** Complete test suite template

### Week 9: Chaos Engineering & Observability
- Fault injection framework design
- OpenTelemetry implementation
- Custom span attributes

- **Project:** Chaos testing framework

### Week 10: Enterprise GTM
- Enterprise sales methodology
- Technical evaluation playbook
- Competitive battle cards

- Partnership strategies
- **Deliverables:** Pitch deck, demo script, 50 prospect list
