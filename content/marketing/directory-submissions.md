**Long Description (under 500 chars):**
Pisama is an open-source platform for detecting and fixing failures in multi-agent AI systems. It identifies 20 failure types including infinite loops, state corruption, persona drift, hallucination, prompt injection, and coordination deadlocks. Using a 5-tier detection pipeline (hash → state delta → embeddings → LLM → human), Pisama achieves 60.1% accuracy on the TRAIL benchmark at $0 cost per trace. Includes a Python SDK (`pip install pisama`), CLI, and MCP server for Cursor/Claude Code. Integrates with LangGraph, CrewAI, AutoGen, n8n, and Dify.
